# Roadmap

## Phase 0 — Local proof, now

- FastAPI control deck.
- Browser VJ output.
- Mock WAV sketch generation.
- Text-only talk-break cues.
- Dry-run stream plans.

## Phase 1 — Clean mix

- Add audio timeline model: intro, body, outro, talk window, crossfade seconds, ducking dB.
- Render continuous local mix with FFmpeg/SoX/Pydub.
- Export cue sheet and waveform preview.

## Phase 2 — Backend adapters

- ACE-Step RunPod endpoint client.
- ComfyUI visual workflow client.
- Local TTS adapter with explicit opt-in.

## Phase 3 — Live routing

- OBS scene/source checklist.
- RTMP dry-run command generator.
- TouchDesigner WebSocket/Spout bridge for Resolume.

## Phase 4 — True autonomous set

- Planner continuously fills queue.
- Energy arc follows user guide.
- Generates talk-breaks between tracks.
- Maintains visual motif and stream status.
