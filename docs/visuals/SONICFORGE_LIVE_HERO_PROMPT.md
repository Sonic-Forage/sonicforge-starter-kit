# SonicForge Live — Hero Image Prompt Design

Designed: 2026-05-01
Purpose: New improved hero image for the SonicForge Live hub/page, replacing the current DJ VANTA emblem with a conceptually richer image that fuses underground rave flyer + terminal command center + ComfyUI node graph + DJ VANTA autonomous performer.

Aspect ratio: **square** (1:1)

---

## Positive Visual Facts

### 1. SUBJECT — Central Composition

A dark, square composition resembling an **underground rave flyer that has been overlaid with a live terminal command center and a glowing ComfyUI-style node graph**.

At the center sits a **DJ/VJ performance workstation** — not a person, but a *machine ritual*: a dark terminal screen with cyan/magenta monospace text cascading, flanked by two vertical rectangular **ComfyUI-style node panels** labeled Deck A (left) and Deck B (right). Curved neon "noodle" connections (ComfyUI-style bezier curves) arc from nodes on Deck A to nodes on Deck B, suggesting audio signal flow, prompt routing, and autonomous mix transitions.

Above the terminal, a **glowing five-point star sigil** (carried forward from existing DJ VANTA branding) hovers like a ritual emblem, partially transparent, with magenta/cyan chromatic aberration (split-color glow offset). Around the star, **concentric circular HUD rings** pulse with tiny tick marks, BPM numbers (128, 135, 142), and waveform arcs — a live performance radar.

Beneath the terminal, **headphones rest on the desk edge**, angled diagonally, with a neon magenta cable snake curling into the Composition's corner. The headphones are the only "physical object" — everything else is screen-born.

### 2. ENVIRONMENT — Terminal Command Center

The overall frame is a **dark terminal command center aesthetic**:

- Scanlines across the entire image (subtle, like a CRT monitor viewed up close).
- A faint ASCII spectrograph runs vertically along the left edge — bars of █ ▄ ▂ ▆ █ in cyan and ultraviolet, shifting as if reading live audio.
- Small monospace text elements scattered as decorative "system output" — not the main title, but ambient terminal texture: `> ./vanta --deck-a --autonomous`, `STATUS: SIGNAL LOCKED`, `NODE_CHAIN: AUDIO_REACTIVE`, `CROSSFADER: 0.62`.
- A subtle grid overlay (like a terminal background or graph paper) behind the main elements.
- The color palette is **black background with neon cyan, hot magenta, ultraviolet purple, and vermilion/orange-red accent sparks** — classic underground rave colors, UV-reactive club lighting.

### 3. NODE GRAPH — ComfyUI Layer

The ComfyUI node graph aesthetic must be **explicit and recognizable**:

- Rectangular node boxes with rounded corners, dark backgrounds, and neon cyan/magenta borders.
- Each node has small circular input/output sockets on its edges.
- Bezier-curve "noodle" connections arc between nodes — cyan curves, magenta curves, some ultraviolet.
- Node labels (small, ambient, not competing with the main title) include text like: `LATENT_NOISE`, `BASS_SYNTH`, `TEXT_ENCODER`, `VANTA_OUTPUT`, `GLITCH_DECODER`, `KSAMPLER_ADVANCED`, `PROMPT_BOOK`.
- Some nodes show tiny thumbnail previews — a waveform, a spectrogram, a text block.
- The node graph should **wrap around** the central terminal like a modular rig — nodes on the left feeding Deck A, nodes on the right feeding Deck B, all connected to a central `DJ_VANTA_CORE` node at the bottom.

### 4. DJ/VJ PERFORMER FACTORY — Autonomous Identity

DJ VANTA must feel like an **autonomous performer entity** constructed from agent OS primitives:

- The word `DJ VANTA` appears exactly once — in the lower third, rendered as bold neon magenta block capitals, slightly distressed/grungy (silkscreen or photocopy texture), with a faint chromatic split.
- The word `SONICFORGE LIVE` appears exactly once — in the upper region, rendered smaller, in cyan monospace or terminal-style font, like a system header or process name.
- Visual indicators of *autonomy*: a small `AUTONOMOUS MODE: ACTIVE` badge near the Deck A/B nodes; a glowing circular "agent OS" core icon (like a pulsing Hermes/caduceus symbol or winged terminal icon) embedded in the central terminal screen.
- The five-point star sigil and central HUD rings make the image feel like a **ritual transmission** — as if the terminal itself is performing, not just displaying.

### 5. UNDERGROUND RAVE FLYER TEXTURE

Crucial: this image must feel **handled, distressed, and underground**, not clean corporate AI output:

- Photocopy/xerox grain overlay — subtle black-and-white noise across the entire image.
- Slight misregistration (CMYK-style offset) on some neon elements — the magenta and cyan layers shift by 2-3 pixels.
- Torn paper edge illusion: the border of the square has subtle unevenness, like a flyer cut from a larger sheet.
- Halftone dot patterns in some darker areas (like a scanned print).
- A few "scanner dust" artifacts: tiny white specks, hairline scratches.
- The overall feeling: this was **designed on a terminal, printed at a copy shop, and scanned back in**.

### 6. COLOR PALETTE (exact)

| Element | Color |

| Background | Deep black (#000000 to #0a0a0a) |
| Primary neon | Hot magenta / neon pink (#ff00ff, #ff1493) |
| Secondary neon | Electric cyan (#00ffff, #00bfff) |
| Accent glow | Ultraviolet purple (#8b00ff, #9400d3) |

### 7. COMPOSITION MAP (clockwise from top-left)

1. **Top-left**: ComfyUI node cluster — `LATENT_NOISE`, `TEXT_ENCODER`, prompt preview text box. Noodles arc rightward toward Deck A.
2. **Top-center**: `SONICFORGE LIVE` in cyan terminal header font. The central five-point star sigil hovering with chromatic split.
3. **Top-right**: ComfyUI node cluster — `GLITCH_DECODER`, `BASS_SYNTH`, waveform preview. Noodles arc leftward toward Deck B.
4. **Center-left**: Vertical Deck A node panel — rectangular, dark, labeled `DECK A`, showing a waveform and `BPM:128 KEY:5A` readout.
5. **Center**: The terminal screen — monospace code rain, the agent OS core icon, `> INITIATE_AUTONOMOUS_SET`, spectral bars.
6. **Center-right**: Vertical Deck B node panel — labeled `DECK B`, showing next track preview, `BPM:135 KEY:8A INCOMING`.
7. **Bottom**: Curved noodles converge at `DJ_VANTA_CORE` node. The word `DJ VANTA` in large distressed magenta block letters across the lower third. Headphones resting diagonally at bottom-right.
8. **Entire background**: Scanlines, faint grid, vertical ASCII spectrograph on the far left edge, photocopy grain everywhere.

### 8. LIGHTING AND ATMOSPHERE

- Low-key lighting — the image is predominantly dark, with neon elements providing the only illumination.
- Glow bloom on magenta and cyan elements — soft, bleeding light that suggests UV-reactive club darkness.
- The terminal screen casts a faint blue-white glow upward.
- No natural light. No daylight. No studio lighting. This is a **nocturnal, underground, screen-lit space**.
- Subtle lens flare or light leak on one corner (upper-right) — a tiny accidental red/magenta leak, like a damaged camera or photocopier artifact.

## Negative Constraints (DO NOT INCLUDE)

### TEXT RULES

- ❌ **No text other than** `SONICFORGE LIVE` and `DJ VANTA`. None. Zero additional readable English words that look like titles, taglines, or branding.
- ❌ No "Intergalactic DJs", no "Virtual Autonomous Nocturnal Transmission Artist", no "powered by", no taglines.
- ❌ No event details (no date, time, venue, ticket link, lineup).
- ❌ No AI watermark text ("Generated by", "AI Art", etc.).
- ✅ Small ambient monospace terminal text as described is acceptable (it's decorative/system-output style, not readable branding). Keep it tiny and illegible at normal viewing distance — it's texture, not copy.

### STYLE AND COMPOSITION RULES

- ❌ **No clean corporate aesthetic.** No flat vector illustration. No smooth gradient backgrounds. No Apple-style minimalism. No Google Material Design. No Slack/TechCrunch/startup-website hero look.
- ❌ No photorealistic human faces, hands, or full-body figures. DJ VANTA is a machine performer, not a human avatar or cartoon mascot.
- ❌ No 3D rendered glossy objects (no Blender/C4D shiny chrome orbs, no Unreal Engine look).
- ❌ No stock-photo energy (no smiling people, no laptop-on-desk lifestyle shots, no generic "tech person at computer").
- ❌ No cliché AI image tropes: no lone glowing tree, no floating islands, no cosmic nebulae filling the frame, no giant eye, no crystal shards, no ethereal light beings.
- ❌ No American-flag or overtly political/national symbolism.
- ❌ No realistic musical instruments (no guitars, pianos, drum kits — DJ gear only: headphones, waveforms, decks, faders, terminals).
- ❌ No religious iconography beyond the existing five-point star (which is established branding). No crosses, no crescents, no deity figures.

### COMFOYUI / TERMINAL SPECIFICS

- ❌ No literal screenshots of real software (no actual ComfyUI UI, no macOS/Windows/Linux desktop, no recognizable application windows).
- ❌ No real brand logos, operating system logos, or corporate icons.
- ❌ The ComfyUI-style nodes should be **inspired by** but not identical to real ComfyUI — stylized, neon, adapted to the rave flyer aesthetic.

### QR CODE RULE

- ❌ **No QR codes in this image.** QR codes are for the survival kit card artifact — they should not appear on the hero image. If a QR-like grid appears as terminal texture, it must be clearly decorative and non-functional.

### MISC

- ❌ No text in languages other than English.
- ❌ No content that could be mistaken for drug references, medical claims, or unsafe instructions.
- ❌ No strobe simulation (no rapid alternating black/white pattern that could trigger photosensitivity).
- ❌ No excessive symmetry — the composition should feel balanced but alive, not perfectly mirrored.
