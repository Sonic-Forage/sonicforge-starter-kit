# Sonic Forage Logo

Primary assets:

- `assets/brand/sonic-forage-banner.png` - recommended GitHub README/org profile banner with embedded strawberry mascot and exact text.
- `assets/brand/sonic-forage-strawberry-avatar.png` - generated AGI strawberry DJ avatar/logo concept.

- `assets/brand/sonic-forage-logo.svg` - README/org-profile banner with exact vector text.
- `assets/brand/sonic-forage-logo.png` - PNG banner fallback.
- `assets/brand/sonic-forage-mark.svg` - square mark/avatar candidate.
- `assets/brand/sonic-forage-mark.png` - PNG square mark/avatar candidate.

## Usage

Use the wide logo at the top of GitHub READMEs, hackathon landing pages, and starter-kit docs. Use the square mark for avatars, favicons, stickers, badges, and social cards.

## Brand notes

- Exact name: **Sonic Forage**
- Wordmark text in assets: **SONIC FORAGE**
- Visual language: neon cyan/magenta/yellow on deep space navy, mushroom/forage + headphones/DJ + waveform signal.
- Keep the logo readable and do not place it over busy backgrounds.

## Safety phrase

The phrase `CLOSED GATES // OPEN SIGNAL` means the project is easy to fork and experiment with, while paid/GPU/public/private-data actions remain closed until explicitly approved.
