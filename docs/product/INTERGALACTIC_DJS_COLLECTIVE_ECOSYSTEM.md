# Intergalactic DJs Collective Ecosystem

Status: launch-readiness product framing for the forkable Party AI OS.

## Naming hierarchy

Lead with the team/collective first:

1. **Intergalactic DJs** — the collective, show, movement, lineup, and public invitation.
2. **DJ VANTA** — the first canonical autonomous performer / first clone.
3. **SonicForge Live** — the universe, runtime, and forkable Party AI Operating System behind the collective.
4. **Hermes** — the agent home for skills, memory, verification, command, and autonomous build loops.
5. **ComfyUI / workflow engines** — operator-armed visual/audio workflow lanes.

Short form:

> Intergalactic DJs is the team. SonicForge Live is the universe. DJ VANTA is the first clone.

## Product model

SonicForge Live should become a forkable ecosystem where builders can:

- clone a local party station,
- create a custom DJ/VJ/persona agent,
- add safe local API/provider config,
- connect ComfyUI workflows,
- add Hermes skills/plugins,
- package crate memories, lore, visual identity, and safety/care copy,
- publish/share agent packs and workflow cards only after secrets and rights are clean,
- eventually train or tune custom models after dataset review, eval planning, license checks, and explicit approval.

## What people should be able to build

- New Intergalactic DJs performer clones.
- Party Mode packs.
- Kandi/PLUR station themes.
- Visual spell workflow packs.
- DJ crate packs and set arcs.
- Hermes skills and adapters.
- Local model/dataset review packs for future Mind Expansion training.

## Safe model-training posture

Training/customization is a future lane, not an automatic launch action. Before any training:

- dataset card exists,
- data rights/consent are clear,
- no private party data or secret material is included,
- eval harness exists,
- base model/license is approved,
- budget/stop condition is approved,
- artifact destination is approved,
- human says yes.

No dataset upload, LoRA/QLoRA/Unsloth/Modal/GPU training, public model card, or private HF upload starts by default.

## Movement line

The elite AI party is invite-only. The builder party is forkable.

Intergalactic DJs is where people stop waiting for permission and start building their own rooms.
