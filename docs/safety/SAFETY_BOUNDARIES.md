# Safety Boundaries

SonicForge starter kits are creative tools with closed gates.

## Closed unless approved

- Paid API calls
- GPU jobs
- model downloads
- model training/fine-tuning
- private media uploads
- public posting/streaming
- bulk messaging
- wallet/financial actions
- cron mutation

## Voice and music rights

- Do not clone named artists' voices without rights.
- Use artists as broad cultural references only; do not claim endorsement.
- Keep datasets private until you intentionally publish them.
- Keep generated media review-only unless the operator approves public release.

## Secrets

Never commit or print real credentials, private endpoints, OAuth files, cookies, SSH private keys, `.env`, or tokens.
