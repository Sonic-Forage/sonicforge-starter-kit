# Contributing

Thanks for helping SonicForge become a forkable creative-agent/DJ starter kit.

## Before opening a PR

```bash
python3 scripts/verify_starter_kit.py
git diff --check
```

## Great contributions

- clearer setup docs
- starter payload examples
- dry-run workflow cards
- frontend launch-cockpit improvements
- verifier/secret-scan hardening
- tutorial issues/discussions for hackathon users

## Do not commit

- `.env` files
- API keys/tokens/private endpoints
- generated private media
- model weights/checkpoints
- workflows that run GPU/paid actions by default
